# SUMSC Documents Page

SUMSC.github.io构建源码仓库。本项目使用 [Vuepress 1.x](https://v1.vuepress.vuejs.org/zh/) 构建。

## 起步

首先你需要拥有`node`和`npm`。

```shell
git clone https://github.com/SUMSC/pages.git
cd pages
npm i
```

### 预览

```shell
npm run home:dev
```

### 构建

```shell
npm run home:build
```

### 部署

```shell
bash ./deploy.sh
```
